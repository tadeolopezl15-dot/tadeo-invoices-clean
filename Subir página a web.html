Perfecto — te explico exactamente cómo subir tu página Next.js (tu app de facturas) a internet paso a paso, sin errores. Vamos directo a lo práctico.

🚀 OPCIÓN RECOMENDADA (LA MEJOR): Netlify o Vercel

Para tu proyecto (Next.js + Stripe + Supabase), lo más estable es:

👉 Vercel (la mejor para Next.js)

🧠 ANTES DE EMPEZAR (OBLIGATORIO)

Tu proyecto debe tener:

✔ Funciona en local (http://localhost:3000)
✔ Tiene .env.local con:

  • Supabase

  • Stripe

  • URL del sitio

Ejemplo importante:

env
NEXT_PUBLIC_SITE_URL=https://tu-dominio.com

⚡ MÉTODO 1 — SUBIR A VERCEL (RECOMENDADO)

🟢 PASO 1: Subir tu código a GitHub

  1. Ve a 👉 GitHub

  2. Crea un repositorio nuevo

  3. En tu proyecto (VS Code terminal):

Bash
git init
git add .
git commit -m "proyecto listo"
git branch -M main
git remote add origin https://github.com/TU-USUARIO/TU-REPO.git
git push -u origin main

🟢 PASO 2: Conectar con Vercel

  1. Ve a 👉 https://vercel.com

  2. Login con GitHub

  3. Click en:

👉 "Add New Project"

  1. Selecciona tu repo

🟢 PASO 3: Configurar variables (MUY IMPORTANTE)

En Vercel → Settings → Environment Variables

Agrega TODO esto:

env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

STRIPE_SECRET_KEY=
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
STRIPE_WEBHOOK_SECRET=

NEXT_PUBLIC_SITE_URL=https://tu-dominio.vercel.app

🟢 PASO 4: Deploy

Click:

👉 Deploy

⏱ En 1–2 minutos tendrás algo así:

https://tu-app.vercel.app

🌐 MÉTODO 2 — CONECTAR TU DOMINIO (Namecheap)

Como tú usas Namecheap:

👉 Namecheap

PASOS:

  1. En Vercel → Settings → Domains

  2. Agrega:

tilequalityimagination.com

  1. En Namecheap → DNS:

Agrega:

Type: A
Host: @
Value: 76.76.21.21

y/o:

CNAME: www → cname.vercel-dns.com

⚠️ ERRORES QUE YA TE ESTÁN PASANDO (y solución)

❌ 404 en páginas

👉 No creaste bien rutas en /app

Ejemplo correcto:

app/
  page.tsx
  invoice/
    page.tsx

❌ localhost funciona pero online no

👉 Te falta:

env
NEXT_PUBLIC_SITE_URL

❌ Stripe no funciona

👉 Webhook mal configurado

Debes usar:

https://tu-dominio.vercel.app/api/stripe/webhook
